# StackPay Monorepo

This repo contains the StackPay web app, API, shared UI primitives, and smart-contract artifacts.

## Structure
- `apps/web`: Next.js 14 UI (App Router)
- `apps/api`: API service (placeholder)
- `packages/ui`: Shared UI components (placeholder)
- `packages/contracts`: Clarity contracts and tests
- `packages/config`: Shared config (placeholder)

## Dev
- `npm run dev` (web)
- `npm run dev:api` (api)
